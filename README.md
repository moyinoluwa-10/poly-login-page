# Poly Login Page

A simple and clean copy login page design for Kogi State Polytechnic.

## Features

- Clean and modern UI design
- Username/Matric number input field
- Password input field with security icon
- "Remember Me" checkbox functionality
- Responsive form layout
- Font Awesome icons integration

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts

## Setup

1. Clone the repository
2. Open `index.html` in your web browser
3. The page is ready to use
